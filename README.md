# Food-Delivery-Route-Optimizer
Food Delivery Route Optimizer is a route optimization system that finds the shortest and most efficient delivery paths using Dijkstra’s Algorithm and Greedy Algorithm concepts.

The project uses graph-based route planning to minimize delivery distance and improve delivery efficiency by selecting optimized paths between locations.

Technologies & Concepts Used
Dijkstra’s Algorithm
Greedy Algorithm
Graph Theory
Web Technologies

Purpose

The purpose of this project is to apply data structures and algorithms to solve real-world delivery route optimization problems.
